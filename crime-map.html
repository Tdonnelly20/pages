Boston's Crime Map

Role: Sole Programmer
Tools: Python
Duration: September-December 2019

Project Status: Complete
Repository: GitHub

Project Description

 As a project for my Big Data class, I programmed an interactive map using python and some of Boston's crime data from 2018. This map separates crimes by color and allows you to move around, zoom in and out, and filter out types of crimes. This resulted in a digital map of Boston with markers that allow you to visualize where certain crimes are happening. If you would like to see my code for this project, it can be found on my GitHub.
